CSS Media Queries

How does a CSS media query work, and why is it useful?

CSS media queries allow you to apply CSS only when the browser or device environment matches a specified rule. This CSS is a key element of responsive web design, allowing you to create different layouts based on the viewport size. It can also be used to detect other information about the environment your site is running in, such as whether the user is using a touchscreen rather than a mouse.
